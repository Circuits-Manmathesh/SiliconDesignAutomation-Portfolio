# Project Evidence Standard

Every project should produce an engineering evidence package, not just a nominal schematic or single waveform.

Required evidence:

- DC characteristics.
- Selected operating point report.
- AC small-signal curves.
- Transient verification curves.
- Pass/fail summary.
- Final dashboard.

The goal is to make each result reviewable by another analog designer. A project should clearly show what was specified, what was selected, what was simulated, what passed, and what remains outside the public scope.
