# Skynet Analog Agent

Skynet Analog Agent is the core physics-aware analog CAD methodology project in this portfolio.

## Methodology

```text
User Requirement
-> Topology Knowledge Pack
-> gm/Id + LUT-Based Device Selection
-> Testbench / Netlist Generation
-> Real Simulation
-> Measurement Extraction
-> Spec Truth Gate
-> Closed-Loop Correction
-> Final Evidence Package
-> Reusable Golden Topology
-> Topology Maturity Map
```

## What It Demonstrates

- topology-aware analog design automation
- gm/Id and LUT-based device selection
- SPICE simulation evidence generation
- measurement extraction
- truth-gated specification evaluation
- closed-loop correction
- reusable golden topology promotion
