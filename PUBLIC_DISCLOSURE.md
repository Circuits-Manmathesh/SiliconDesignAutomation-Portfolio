# Public Disclosure

This repository presents an open, educational, non-confidential analog/RF/mixed-signal design methodology portfolio.

It does not include:

- proprietary PDK files
- foundry model files
- confidential client data
- commercial IP
- internal company reports
- restricted technology files
- private scripts
- protected customer information
- NDA material

All diagrams, notes, plots, tables, screenshots, and reports are sanitized public portfolio artifacts.

The portfolio is intended to explain circuit understanding, CAD methodology, automation architecture, and public-safe design evidence.
